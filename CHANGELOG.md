# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial project setup with Turborepo monorepo structure
- NestJS API backend with MongoDB
- React frontend with Vite
- Shared packages for types, schemas, and UI components
- User authentication with JWT
- Role-based access control (Admin, Dispatcher, Field Responder, Viewer)
- Event management system with CRUD operations
- Status tracking for field responders
- Real-time WebSocket communication
- Interactive map with Leaflet
- Geospatial queries for nearby events
- Responsive dashboard with real-time statistics
- User management interface
- Event creation and editing forms
- Docker Compose setup for MongoDB
- Comprehensive documentation (README, API docs, Architecture, Development guide)

### Security
- JWT-based authentication
- Password hashing with bcrypt
- Role-based access control
- CORS configuration
- Input validation with Zod schemas
